# Install all env 

This script help us to install all the enviroment 
```sh
wget https://raw.githubusercontent.com/uracilo/installenv/main/script.sh
```

```sh
chmod +x ./script.sh
./script.sh
```

sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
